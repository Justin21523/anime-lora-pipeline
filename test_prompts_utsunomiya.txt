## 測試參數建議 (Recommended Settings)

**基礎設定:**
- Steps: 20-30
- Sampler: DPM++ 2M Karras 或 Euler a
- CFG Scale: 7-9
- Size: 512x512 或 512x768
- LoRA Weight: 0.7 ~ 1.0

**如果角色特徵不明顯:**
- 提高 LoRA 權重到 0.9-1.0
- 增加 "orange hair" 和 "utsunomiya_toramaru" 的權重

**如果過擬合/太僵硬:**
- 降低 LoRA 權重到 0.6-0.7
- 嘗試不同的 checkpoint (epoch 6 或 8)

**檢查點比較測試:**
1. utsunomiya_toramaru_v1.safetensors (Epoch 10 - 最終版)
2. utsunomiya_toramaru_v1-000008.safetensors (Epoch 8)
3. utsunomiya_toramaru_v1-000006.safetensors (Epoch 6)

## 品質指標 (Quality Indicators)

**良好結果:**
✓ 橘色刺蝟頭髮型清晰
✓ 面部特徵一致
✓ 足球服裝正確
✓ 姿勢自然
✓ 無明顯解剖錯誤

**需要調整:**
✗ 頭髮顏色錯誤
✗ 面部崩壞
✗ 服裝錯誤
✗ 姿勢僵硬或扭曲
✗ 手部/腳部變形

**改善方法:**
- 調整 LoRA 權重
- 更換 checkpoint 版本
- 增加負面提示詞
- 調整 CFG Scale
- 使用 Hires.fix 提升細節
